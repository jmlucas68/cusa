===================================================
 Proyecto: Página Web Estática sobre Nicolás de Cusa
===================================================

Descripción del Proyecto
------------------------
Este proyecto consiste en un único archivo HTML (`index.html`) que crea una página web estática, visualmente atractiva y completa sobre la vida, obra y pensamiento del filósofo y teólogo Nicolás de Cusa (1401-1464).

El objetivo es presentar la información de manera clara, estructurada y fácil de navegar, sirviendo como un recurso educativo autocontenido. El diseño está inspirado en la página sobre Platón de jmlucas68.


Características
---------------
*   **Diseño Visual Atractivo:** Uso de tipografías elegantes (Lato y Playfair Display), una paleta de colores sobria y un diseño basado en tarjetas para una lectura agradable.
*   **Single-Page:** Todo el contenido, estilos y estructura están en un solo archivo HTML, lo que facilita su distribución y visualización sin necesidad de un servidor web.
*   **Navegación Fluida:** Un menú de navegación fijo en la parte superior permite desplazarse suavemente entre las diferentes secciones de la página.
*   **Contenido Estructurado:** La información se organiza en las siguientes secciones:
    1.  **Biografía:** Un resumen de la vida y carrera de Nicolás de Cusa.
    2.  **Contexto Histórico y Filosófico:** Panorama del siglo XV para entender su figura.
    3.  **Pensamiento:** Explicación de sus ideas clave como la "Docta Ignorancia" y la "Coincidencia de los Opuestos", ilustradas con citas de sus obras.
    4.  **Influencias:** Un análisis de las corrientes filosóficas que lo formaron y el impacto que tuvo en pensadores posteriores.
    5.  **Obras Principales:** Un listado de sus textos más importantes con enlaces para su adquisición o consulta.
    6.  **Recursos Externos:** Enlaces a fuentes académicas como la Stanford Encyclopedia of Philosophy y búsquedas en Academia.edu para fomentar una investigación más profunda.


Tecnología Utilizada
--------------------
*   **HTML5:** Para la estructura semántica del contenido.
*   **CSS3:** Para el diseño, la maquetación y la interactividad visual. Los estilos están incluidos dentro del propio archivo HTML en una etiqueta `<style>`.
*   **Google Fonts:** Para mejorar la estética tipográfica.


Cómo Utilizar
-------------
1.  Guarda el código HTML proporcionado en un archivo con el nombre `index.html`.
2.  Abre el archivo `index.html` directamente en cualquier navegador web moderno (como Google Chrome, Mozilla Firefox, Microsoft Edge, etc.).
3.  No se requiere conexión a internet para ver la estructura y el contenido base, aunque sí para cargar las fuentes de Google y acceder a los enlaces externos.


Créditos e Inspiración
----------------------
*   El diseño y la estructura de la página están inspirados en el excelente trabajo de https://jmlucas68.github.io/platon/
*   El contenido ha sido generado y estructurado por una IA, basándose en fuentes filosóficas de referencia.
